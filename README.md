# 🥟 Momos Hub • 3D Cart Website

A stunning, interactive 3D website for a momos and cold coffee cart. Built with **Three.js** and **GSAP** for smooth animations, featuring the official Momos Hub color palette: Spiced Orange & Iced Teal.

![Momos Hub Preview](https://via.placeholder.com/800x400/FF7A2F/FFFFFF?text=Momos+Hub+3D+Preview)

## ✨ Features

- **Immersive 3D Scene**: Stylized momos steamer and cold coffee glass with realistic lighting.
- **Smooth Animations**: GSAP-powered entrance, floating, and scroll-based effects.
- **Interactive Elements**: Orbit controls to view products from any angle, buttons with playful feedback.
- **Fully Responsive**: Works on desktop and mobile devices.
- **Brand Colors**: Warm orange (`#FF7A2F`) and cool teal (`#00B4B3`) applied throughout.
